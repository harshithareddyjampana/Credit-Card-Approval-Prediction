# Credit Card Approval Prediction

## Project Overview

The Credit Card Approval Prediction project aims to predict whether a credit card application should be approved or rejected using machine learning techniques.

## Dataset Source

The dataset was collected from Kaggle's Credit Card Approval Prediction dataset.

Files used:

* application_record.csv
* credit_record.csv

## Epic 1: Data Collection

The dataset was downloaded, extracted, and prepared for further analysis and machine learning model development.

## Objective

To analyze customer demographic and financial information and build a model that can predict credit card approval status.
